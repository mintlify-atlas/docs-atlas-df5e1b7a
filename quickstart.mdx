This guide will help you get the Hotel Reservation System up and running quickly. You'll learn how to install dependencies, configure the database, start the server, and make your first API call.

<Steps>

<Step title="Install Dependencies">

First, ensure you have Node.js installed on your system. Then clone the repository and install the required packages:

```bash
cd reservaionFronEnd
npm install
```

This will install the following core dependencies:
- **express** - Web framework for handling HTTP requests
- **mysql2** - MySQL database driver
- **body-parser** - Middleware for parsing request bodies
- **cors** - Cross-Origin Resource Sharing support

</Step>

<Step title="Set Up MySQL Database">

Create a MySQL database for the hotel reservation system:

```sql
CREATE DATABASE BDReservacion;
USE BDReservacion;

-- Create cliente table
CREATE TABLE cliente (
  id INT AUTO_INCREMENT PRIMARY KEY,
  nombre VARCHAR(255) NOT NULL,
  correo VARCHAR(255) NOT NULL,
  telefono VARCHAR(20)
);

-- Create habitaciones table
CREATE TABLE habitaciones (
  id INT AUTO_INCREMENT PRIMARY KEY,
  nombre VARCHAR(255) NOT NULL,
  tipo VARCHAR(100) NOT NULL,
  precio DECIMAL(10, 2) NOT NULL
);

-- Create reservaciones table
CREATE TABLE reservaciones (
  id INT AUTO_INCREMENT PRIMARY KEY,
  folio VARCHAR(100) NOT NULL,
  nombrecliente VARCHAR(255) NOT NULL
);
```

<Tip>
For detailed database setup instructions, see the [Database Setup](/deployment/database-setup) guide.
</Tip>

</Step>

<Step title="Configure Database Connection">

The database connection is configured in `model/database.js`. Update the connection settings if needed:

```javascript model/database.js
const mysql = require('mysql2');

const conexion = mysql.createConnection({
    host: 'localhost',
    user: 'root',
    password: '1234',
    database: 'BDReservacion'
});

conexion.connect((err) => {
    if (err) throw err;
    console.log("Conexion exitosa a la base de datos")
});

module.exports = conexion;
```

<Warning>
Make sure to update the `user` and `password` fields to match your MySQL credentials.
</Warning>

</Step>

<Step title="Start the Server">

Run the Express server:

```bash
node server.js
```

You should see:
```
Servidor iniciado en el puerto 4000
Conexion exitosa a la base de datos
```

The API is now running at `http://localhost:4000`

</Step>

<Step title="Make Your First API Call">

Test the API by creating a new client:

<CodeGroup>

```bash cURL
curl -X POST http://localhost:4000/clientes \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Juan Pérez",
    "correo": "juan@example.com",
    "telefono": "555-0123"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:4000/clientes', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    nombre: 'Juan Pérez',
    correo: 'juan@example.com',
    telefono: '555-0123'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    'http://localhost:4000/clientes',
    json={
        'nombre': 'Juan Pérez',
        'correo': 'juan@example.com',
        'telefono': '555-0123'
    }
)

print(response.json())
```

</CodeGroup>

Expected response:
```json
{
  "message": "Cliente agregado"
}
```

</Step>

<Step title="Retrieve All Clients">

Verify the client was created:

const response = await fetch('http://localhost:4000/clientes');

</Step>

</Steps>

## Access the Web Interface

Open the frontend in your browser:

1. Navigate to `reservacionBackEnd/index.html` to manage clients
2. Navigate to `reservacionBackEnd/habitaciones.html` to manage rooms
3. Navigate to `reservacionBackEnd/reservaciones.html` to manage reservations

<Note>
Make sure the backend server is running on port 4000 before accessing the web interface.
</Note>

## Common Issues

<Accordion title="Cannot connect to MySQL database">
  
**Error:** `Error: Access denied for user 'root'@'localhost'`

**Solution:** Update your MySQL credentials in `model/database.js` or create a new MySQL user:

```sql
CREATE USER 'hoteluser'@'localhost' IDENTIFIED BY 'yourpassword';
GRANT ALL PRIVILEGES ON BDReservacion.* TO 'hoteluser'@'localhost';
FLUSH PRIVILEGES;
```

</Accordion>

<Accordion title="Port 4000 already in use">

**Error:** `Error: listen EADDRINUSE: address already in use :::4000`

**Solution:** Either stop the process using port 4000 or change the port in `server.js`:

```javascript
const PORT = 5000; // Change to an available port
```

</Accordion>

<Accordion title="CORS errors in browser">

**Error:** `Access to fetch at 'http://localhost:4000/clientes' from origin 'null' has been blocked by CORS policy`

**Solution:** CORS is already configured in `server.js`. Make sure you're running the server and accessing the frontend through a web server (not directly opening HTML files).

</Accordion>
