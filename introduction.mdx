## Architecture Overview

The system follows a clean MVC architecture:

- **Backend**: Node.js with Express.js framework
- **Database**: MySQL for data persistence
- **Frontend**: Vanilla JavaScript with Bootstrap
- **API**: RESTful endpoints with CORS support

<Note>
  This system is designed for small to medium-sized hotels looking for a straightforward management solution.
</Note>

## Quick Example

Get started with a simple API call to fetch all clients:

```bash
curl http://localhost:4000/clientes
```

Response:

```json
[
  {
    "id": 1,
    "nombre": "Juan Pérez",
    "correo": "juan@example.com",
    "telefono": "555-0123"
  }
]
```
