## Overview

The Client Management module provides a comprehensive interface for managing hotel guest information. Staff can create, view, update, and delete client records, maintaining essential contact details for all hotel guests.

<Note>
  All client operations are performed through a RESTful API running on `http://localhost:4000/clientes`
</Note>

## User Interface

The client management interface (`index.html`) displays a table-based view of all registered clients with the following information:

- **ID**: Unique identifier for each client
- **Name (Nombre)**: Client's full name
- **Email (Correo)**: Contact email address
- **Phone (Teléfono)**: Contact phone number
- **Actions (Acciones)**: Quick access to update or delete operations

## Managing Clients

<Steps>
  <Step title="View All Clients">
    When you navigate to the Clients page, the system automatically loads all registered clients from the database. The table displays client information in an organized format with action buttons for each record.

    ```javascript
    // Data is loaded on page initialization
    async function loadDataCliente() {
        const response = await fetch(URL);
        data = await response.json();
        // Renders client data into the table
    }
    ```
  </Step>

  <Step title="Add a New Client">
    Click the **"Nuevo Cliente"** (New Client) button at the top of the page to open the creation modal.

    <Accordion title="Client Creation Form">
      The modal form includes three required fields:

      - **Nombre** (Name): Client's full name
      - **Correo** (Email): Valid email address
      - **Teléfono** (Phone): Contact phone number

      After filling out the form, click **"Guardar"** (Save) to create the client record.
    </Accordion>
  </Step>

  <Step title="Update Client Information">
    Each client row has an **"Actualizar"** (Update) button. Clicking it opens a dialog pre-filled with the current client information.

    The update dialog uses the same fields as the creation form, allowing you to modify:
    - Name
    - Email
    - Phone number

    Click **"Actualizar"** (Update) to save changes or **"Cerrar"** (Close) to cancel.
  </Step>

  <Step title="Delete a Client">
    Click the **"Eliminar"** (Delete) button next to any client to remove them from the system. This action immediately removes the client record from the database.

    <Note>
      Client deletion is permanent. Ensure the client has no active reservations before deleting.
    </Note>
  </Step>
</Steps>

## API Operations

The client management system implements full CRUD operations through the following API endpoints:

<Tabs>
  <Tab title="Create Client">
    ```javascript
    async function guardarDataCliente() {
        const dataCliente = {
            nombre: document.getElementById("clientName").value,
            correo: document.getElementById("clientEmail").value,
            telefono: document.getElementById("clientPhone").value
        };

        const response = await fetch(URL, {
            method: "POST",
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify(dataCliente)
        });

        if (!response.ok) throw new Error('Error al guardar cliente');
        loadDataCliente();
    }
    ```

    **Endpoint**: `POST /clientes`

    **Payload**:
    ```json
    {
      "nombre": "John Doe",
      "correo": "john.doe@example.com",
      "telefono": "555-1234"
    }
    ```
  </Tab>

  <Tab title="Read Clients">
    ```javascript
    async function loadDataCliente() {
        const response = await fetch(URL);
        data = await response.json();

        let html = '';
        data.forEach(cliente => {
            html += `
            <tr>
                <td>${cliente.id}</td>
                <td>${cliente.nombre}</td>
                <td>${cliente.correo}</td>
                <td>${cliente.telefono}</td>
                <td>
                    <button onclick="modificarDataCliente(${cliente.id})">Actualizar</button>
                    <button onclick="deleteDataCliente(${cliente.id})">Eliminar</button>
                </td>
            </tr>`;
        });

        T_DATA_CLIENTE.innerHTML = html;
    }
    ```

    **Endpoint**: `GET /clientes`

    **Response**:
    ```json
    [
      {
        "id": 1,
        "nombre": "John Doe",
        "correo": "john.doe@example.com",
        "telefono": "555-1234"
      }
    ]
    ```
  </Tab>

  <Tab title="Update Client">
    ```javascript
    async function actualizarDataCliente() {
        const dialog = document.getElementById("formDataCliente");
        const id = dialog.dataset.clienteId;

        const dataCliente = {
            nombre: document.getElementById("clientNameU").value,
            correo: document.getElementById("clientEmailU").value,
            telefono: document.getElementById("clientPhoneU").value
        };

        const response = await fetch(`${URL}/${id}`, {
            method: "PUT",
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify(dataCliente)
        });

        if (!response.ok) throw new Error('Error al actualizar cliente');
        
        dialog.close();
        loadDataCliente();
    }
    ```

    **Endpoint**: `PUT /clientes/:id`

    **Payload**:
    ```json
    {
      "nombre": "Jane Doe",
      "correo": "jane.doe@example.com",
      "telefono": "555-5678"
    }
    ```
  </Tab>

  <Tab title="Delete Client">
    ```javascript
    async function deleteDataCliente(id) {
        const response = await fetch(`${URL}/${id}`, { method: "DELETE" });
        if (!response.ok) throw new Error('Error al eliminar cliente');
        
        loadDataCliente();
    }
    ```

    **Endpoint**: `DELETE /clientes/:id`
  </Tab>
</Tabs>

### Update Dialog

A native HTML `<dialog>` element (`#formDataCliente`) handles client updates:

```javascript
function modificarDataCliente(id) {
    const dataCliente = data.find(cliente => cliente.id === id);
    document.getElementById("clientNameU").value = dataCliente.nombre;
    document.getElementById("clientEmailU").value = dataCliente.correo;
    document.getElementById("clientPhoneU").value = dataCliente.telefono;

    const dialog = document.getElementById("formDataCliente");
    dialog.dataset.clienteId = id;
    dialog.showModal();
}
```

## Error Handling

All API operations include comprehensive error handling:

```javascript
try {
    const response = await fetch(URL, { /* options */ });
    if (!response.ok) throw new Error('Error message');
    // Success operations
} catch (error) {
    console.error(error);
}
```

Errors are logged to the browser console for debugging purposes. In production, these should be displayed to users through appropriate UI feedback mechanisms.
